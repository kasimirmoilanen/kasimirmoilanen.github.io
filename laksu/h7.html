Kirjoitin Hello World -ohjelman kolmella kielellä käyttäen apunani internetin lähteitä (Lähteet sivun lopussa).

Valintani olivat bash, C++ ja Python.

Bash:

echo "Hello World!"

C++:

int main() {
std::cout << "Hello, C++ World!" << std::endl;
return 0;
}

Python:

print("Hello world!")

Testasin että kaikki toimivat:

If you create a script that everybody should be able to run copy it to /usr/local/bin/
